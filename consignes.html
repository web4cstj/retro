Exercice Retro - Consignes

  • Préparation
    1. Télécharger le dossier retro et le mettre sur le disque D:. Il s’agit de l’application Web que vous devez faire.
    2. Installer cette application sur Wampserver en créant son alias.
    3. Changer l’extension des fichiers .htm en .php.
    4. Visualiser la page index.php dans Firefox en passant par localhost. Elle devrait bien s’afficher malgré le changement d’extension de fichier.
    5. Ouvrir la page index.php dans Dreamweaver ou Notepad++.
    6. Ouvrir également TOUS les fichiers du dossier sections.
  • Modification
    1. Trouver la portion du code HTML qui désigne le logo avec son lien.
    2. Copier et coller ce code dans le fichier logo.inc.php.
    3. Remplacer ce code HTML dans index.php par la fonction include, qui pointe sur logo.inc.php. Ne pas oublier l’ouverture des codes PHP.
    4. Sauvegarder les 2 fichiers et visualiser les changements dans Firefox.
    5. Répéter l’opération pour les fichiers titre.inc.php, pub.inc.php, menu.inc.php, pied.inc.php et finalement entete.inc.php.
    6. Remarquez la simplification du code HTML et appliquez les changements aux autres pages du site.
  • Particularités
    1. Si l’opération a bien été faite, le fichier entete.inc.php devrait contenir également des commandes include. C’est très normal.
    2. On doit modifier les liens du menu et du logo afin de refléter les changement d’extension de fichiers.
    3. Remettre l’exercice sur REMISE une fois terminé.
